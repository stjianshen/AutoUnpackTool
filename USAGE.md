# 自动解压工具 - 使用指南

## 功能概述

这是一个基于 C# WPF 开发的自动解压工具，支持自动测试密码并解压加密的压缩文件。

## 主要功能

### 1. 软件设置

点击界面上的 **"软件设置"** 按钮，可以配置以下选项：

- **解压软件路径 (7z.exe)**：必须设置！
  - 需要下载并安装 7-Zip
  - 找到 7-Zip 安装目录下的 `7z.exe` 文件
  - 默认安装路径通常在：`C:\Program Files\7-Zip\7z.exe`
  
- **解压后原文件处理**：
  - 保留原文件（默认）
  - 移动到回收站
  - 直接删除

- **并发线程数**：1-8（默认2）

- **密码本文件路径**（可选）：
  - TXT 格式，每行一个密码
  - 如果不设置，使用默认密码本

### 2. 使用方法

1. **配置 7z.exe 路径**（首次使用必须）：
   - 点击 **"软件设置"** 按钮
   - 浏览选择 7z.exe 文件
   - 点击 **"保存"**

2. **选择输出目录**：
   - 点击 **"选择目录"** 按钮
   - 选择解压文件的保存位置

3. **添加压缩文件**：
   - 直接拖拽压缩文件到窗口
   - 支持批量拖拽多个文件

4. **开始解压**：
   - 如果勾选了 **"自动解压"**，拖拽文件后会自动开始
   - 或者点击 **"开始解压"** 按钮手动开始

### 3. 密码测试

程序会自动尝试以下密码：
- 默认密码：123456, password, 12345678, admin, root
- 自定义密码本（在设置中配置）

### 4. 状态查看

- **左侧列表**：显示文件处理状态
- **右侧日志**：详细的处理日志输出
- **底部状态栏**：显示当前测试的密码和线程状态

## 7-Zip 下载安装

1. 访问官网：https://www.7-zip.org/
2. 下载 Windows 版本（推荐 64 位 x64）
3. 安装后找到 `7z.exe` 的路径
4. 在软件设置中配置该路径

## 配置文件位置

程序配置保存在：
```
%APPDATA%\AutoUnpackTool\settings.json
```

## 注意事项

- ⚠️ **必须配置 7z.exe 路径才能使用解压功能**
- 首次使用请先在"软件设置"中完成配置
- 大文件解压可能需要较长时间，请耐心等待
- 建议在测试时使用不重要的文件
- 支持多种压缩格式：ZIP, RAR, 7Z, TAR, GZ, BZ2 等（取决于 7-Zip 支持）

## 常见问题

**Q: 为什么无法解压？**
A: 请检查：
1. 是否已正确配置 7z.exe 路径
2. 输出目录是否有写入权限
3. 压缩文件是否损坏

**Q: 如何添加更多密码？**
A: 在"软件设置"中指定密码本文件路径，创建一个 TXT 文件，每行一个密码。

**Q: 支持哪些压缩格式？**
A: 支持 7-Zip 支持的所有格式，包括：7Z, ZIP, RAR, CAB, GZIP, BZIP2, TAR, LZMA, XZ 等。

## 技术说明

- 开发框架：.NET 10.0 WPF
- 解压引擎：7-Zip 命令行工具
- 异步处理：使用 async/await 避免界面卡顿
- 配置存储：JSON 格式保存在 AppData 目录
